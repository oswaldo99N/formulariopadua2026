# Formulario de Retiro - Colegio San Antonio de Padua

Este proyecto es una aplicación web moderna (React + Vite) diseñada para gestionar las inscripciones a retiros espirituales de una manera intuitiva y original.

## Características

* **Diseño "Wizard"**: Formulario paso a paso para evitar abrumar al usuario.
* **Temática Espiritual**: Colores y estilos cuidadosamente seleccionados (Azul profundo, Dorado, Terracota).
* **Efecto Glassmorphism**: Estética moderna y elegante.
* **Gestión Legal Completa**: Secciones claras para normas de comportamiento y exoneración de responsabilidad.

## Cómo Iniciar

1. Instalar dependencias:

    ```bash
    npm install
    ```

2. Iniciar el servidor de desarrollo:

    ```bash
    npm run dev
    ```

3. Abrir en el navegador:
    `http://localhost:5173`

## Estructura del Proyecto

* `src/components/WizardForm.jsx`: Componente principal que gestiona los pasos.
* `src/components/steps/`: Componentes individuales para cada sección del formulario.
* `src/styles/`: Archivos CSS (main.css) con las variables de diseño.
